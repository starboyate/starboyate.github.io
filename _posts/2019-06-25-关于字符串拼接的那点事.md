## 二、场景分析
##### 1. 场景1
```java
public class Main {
    public static void main(String[] args) {
        String a = "1" + "2";
        StringBuilder sb = new StringBuilder();
        sb.append("1");
        sb.append("2");
        StringBuffer buffer = new StringBuffer();
        buffer.append("1");
        buffer.append("2");
    }
}
```
